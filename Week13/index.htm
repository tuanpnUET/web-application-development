Lab. Định tuyến URL

Mục đích

Phát triển module định tuyến URL thực hiện ánh xạ các REST API đến controller.action.

Nội dung

Download mã nguồn

Yêu cầu

- Phân tích mã nguồn cài đặt bộ điều khiển mặt trước (tệp index.php)
- Phân tích mã nguồn cài đặt bộ định tuyến (tệp app/core/control/Router.php). Với mỗi ánh xạ URI sang controller.action (API), đọc mã nguồn cài đặt API tương ứng rồi sử dụng trình duyệt để truy cập API đó:
+ GET /students (Sử dụng tệp list.htm để làm frontend)
+ GET /students/{id} (Sử dụng tệp get.htm để làm frontend)
+ PUT /students/{id} (Sử dụng tệp update.htm để làm frontend)
- Mở rộng trang update.htm cho phép người dùng nhập thêm các thông tin ngày sinh, giới tính, quê quán. Đồng thời, nâng cấp lớp StdController và phương thức updateOneStd() đọc dữ liệu của người dùng đã PUT lên và lưu dữ liệu vào các thuộc tính của lớp StdController.
- Nâng cấp phương thức getByID() của StdController để trả về dữ liệu từ các thuộc tính của lớp StdController.
- Truy cập update.htm để nhập dữ liệu rồi truy cập get.htm để quan sát dữ liệu đã nhập.
